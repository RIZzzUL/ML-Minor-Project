# ML-Minor-Project
# 💳 German Credit Risk Prediction

This project uses Machine Learning to predict credit risk (High/Low) based on customer financial data.

## 📊 Project Features
* **EDA:** Explored data trends using Seaborn and Matplotlib.
* **Modeling:** Implementation of classification algorithms for risk assessment.
* **Web Interface:** Interactive app built with **Streamlit**.

## 📁 Repository Contents
* `app.py`: Streamlit application script.
* `MI Minor Project (1).ipynb`: Full data analysis and training notebook.
* `german_credit_data 3.csv`: Dataset used for training.
* `requirements.txt`: Python dependencies for cloud deployment.
